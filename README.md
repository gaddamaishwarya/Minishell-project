The MiniShell Project is a custom command-line interface developed in C that replicates the core functionality of a Linux shell. It interprets user input, parses commands, and executes both built-in and external programs by directly interacting with the operating system. The shell features a custom prompt (MiniShell:$) and supports essential built-in commands such as pwd, cd, echo, exit, and clear. It also implements special shell variables like $$ (process ID), $? (exit status of the last executed command), and $SHELL (path of the shell). In addition to built-in functionality, the MiniShell can execute external commands like ls, sleep, and date using system calls. The project is built using core system programming concepts including process creation with fork(), command execution with execvp(), and basic command parsing logic. This project provided practical exposure to how operating systems handle command execution internally and helped strengthen skills in C programming, debugging, and low-level system design.
